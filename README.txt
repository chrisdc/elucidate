Elucidate
=========

A simple, clean theme with an emphasis on typography and white space. Elucidate is a fully responsive, mobile first theme based on Underscores.

Front Page Template
===================

Elucidate includes a template front-page.php designed to serve as an optional front page to your website. This includes a full-width featured image and content area, as well as 3 widget areas arranged horizontally along the page. To use this template create a page as normal, but under template choose 'Front Page Template'

Social Media Menu
=================

Eludiate includes a menu in the footer to display links to your social media accounts. To use this follow the following steps.

1. On your dashboard open the 'Edit Menus' screen (Appearance > Menus) and click 'create a new menu'
2. Under theme location tick the box 'Social Media Menu'
3. Add links to your social media accounts
4. Click 'Save Menu' to finish.

This menu currently supports the following social media links:

* Dribbble
* Email (Any mailto: link)
* Facebook
* Flickr
* GitHub
* Google+
* Pinterest
* RSS
* Twitter
* WordPress (wordpress.com sites or links to .org profiles)
* YouTube

Specifications
==============

The standard column width is 640px
The content width on front-page.php is 920px
